An Introduction to Bludit

Surely anyone reading this is already aware that Bludit is a portable, flat-file content management system for building websites. But when first installing Bludit you may be surprised to learn that it does not have as much built-in functionality as something like WordPress. Although Bludit is a decent CMS, the default themes offered by Bludit are not attractive, and the default plugins leave much to be desired.

Visit the Bludit website at www.bludit.com.
